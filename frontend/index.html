System Features
🔒 SHA-256 Hashing

Every block contains a cryptographic hash linking to the previous block, making tampering computationally infeasible.

🔑 EC Cryptography

Wallets use elliptic curve secp256k1 key pairs — the same standard used by Bitcoin and Ethereum networks.

🛡 Threat Analysis

Real-time transaction risk scoring with ML-style pattern detection to flag suspicious activities before confirmation.

⛏ Proof of Work

Blocks are mined with configurable difficulty, requiring computational effort to add new records to the chain.

🔐 JWT Auth

All protected routes require signed JWT tokens. Passwords are bcrypt hashed with 12 salt rounds.

⚡ Rate Limiting

API endpoints are protected with rate limiting and Helmet.js security headers to prevent abuse attacks.
